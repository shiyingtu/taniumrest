Run Query Guidelines

  • Two modes of operation are supported for the run query action:
    • Saved Questions
      • User can save a manually created question on the Tanium server, then provide the name of that saved question in the query_text parameter to fetch appropriate results.
      • The is_saved_question parameter has to be set to True for this to work correctly.
    • Manual Questions
      • Users can directly provide the question to be asked to the Tanium server in the query_text parameter to fetch appropriate results.
      • The is_saved_question parameter has to be set to False for this to work correctly.
      • To run the query on particular computers, users can create a computer group with the IP address/hostname of the target computers and can specify that group name in the group_name parameter.
      • If group_name parameter is not provided, then the query will be executed on all the registered IP addresses/hostnames.

Terminate Process Guidelines

  • Please follow the steps below to execute this action successfully:
    • Create and save a package on the Tanium server with a meaningful package name and add a command to terminate the required process in the package's command section.
    • To terminate the process of particular computers, users can create a computer group with the IP address/hostname of the target computers and can specify that group name in the group_name parameter.
    • If group_name parameter is not provided, then the terminate process action will be executed on all the registered IP addresses/hostnames.
